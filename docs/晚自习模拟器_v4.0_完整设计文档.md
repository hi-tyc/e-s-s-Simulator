# 文档导读

### 核心价值主张
> **《晚自习模拟器》不是"让学生玩游戏逃避晚自习"，而是"让学生在游戏中体验晚自习，从而理解自己"。**

中国青少年抑郁检出率 **24.6%**（中科院心理所 2023）。晚自习（18:30-21:30）是青少年一天中压力峰值时段，但学校、家长、甚至学生自己，都把这种压力视为"正常"。**没有人教青少年如何识别、表达、管理这种压力。**

本游戏是一个**同理心引擎**——让不同立场的人，在体验中理解彼此。

### 关键设计约束
- **视角**：第一人称；未获准时以固定座位感官管理为核心，举手获准后可进入 60 秒自由活动（鼠标控制朝向、空格前进、Shift 侧身），通过前门/后门进入走廊。课间阶段允许 5 分钟自由活动，可去饮水机、洗手间或一班前方左转门厅看向户外。
- **身体需求**：拆分为口渴、饥饿、如厕三项，另设水杯容量。喝水会降低口渴并消耗杯水，饮水机可补满水杯，洗手间交互会消耗 10 秒并清空如厕需求。
- **视野**：约水平100度，垂直70度，有边缘模糊和景深
- **NPC数量**：20人（从50人削减，聚焦深度互动）
- **美术风格**：Low Poly程序化生成，纯色+简单贴图
- **信息获取**：70%音频驱动 + 20%视觉余光 + 10%UI提示
- **精神内核**：心理真实性优先 > 物理真实性优先

# 第一部分：精神内核与价值化设计系统

## 第一章：价值主张与心理机制

### 1.1 游戏解决什么问题？

**核心问题：中国高中生的心理健康危机被系统性忽视。**

- 高中生不知道自己正在经历什么。焦虑、孤独、同伴压力、对规则的反叛——这些感受被压抑为"我不够努力"或"我想太多"。
- 教师和家长看到学生"不认真学习"，但看不到学生正在经历的**心理和社会压力**。
- 中国的晚自习制度是"默认设置"，很少有人质疑它是否合理。

**游戏如何解决**：
- **内心独白系统**：玩家听到角色的真实想法（"我好累...但不敢停下来"）
- **状态可视化**：心理能量、情绪调节能力、社会面具成本——不是隐藏属性，而是**可见的、需要管理的资源**
- **"崩溃"事件**：不是Game Over，而是**教育时刻**——游戏暂停，弹出"你的角色感到不堪重负。这不是失败，这是信号。"
- **多结局反思**：结局不是"你赢了/输了"，而是"你选择了A，这意味着你重视X。其他学生选择了B，他们重视Y。没有标准答案。"

### 1.2 三个层面的社会价值

#### 1.2.1 学生层面：情绪识别与自我认知（核心）

**价值**：学生通过游戏学会**识别自己的情绪模式**，"原来我晚自习摸鱼不是因为懒，是因为我真的需要休息"。

#### 1.2.2 教师/家长层面：同理心建设（双重视角的核心）

**价值**：教师和家长通过游戏体验**权力不对等下的生存策略**，从而减少对学生的道德评判，增加理解。

- **教师视角**：玩家扮演老师，发现"管理50个学生是不可能的"，理解为什么必须"选择性执法"。更重要的是，发现**学生的违规行为背后有原因**——那个传纸条的学生可能在求救，那个睡觉的学生可能昨晚打工到两点。
- **"真相"系统**：教师视角通关后，解锁"学生视角回放"——看到学生一整晚的内心戏，理解"原来他不是在反抗我，他是在崩溃边缘"。

#### 1.2.3 社会层面：公共讨论与教育反思

**价值**：游戏成为一个**社会实验场**，让玩家（包括未来的教育工作者、政策制定者）体验制度如何塑造行为。

- **数据层**：游戏结束后显示真实数据（"你今晚经历了3次焦虑峰值。中国高中生平均每晚经历5.2次。"）
- **"如果..."模式**：沙盒模式允许玩家调整参数——"如果晚自习只有1小时？""如果允许说话？""如果取消排名？"——让玩家自己发现制度设计如何影响行为。
- **叙事出口**：某些结局触发"真实故事"——基于真实高中生访谈的短篇文本。

### 1.3 设计支柱（价值增强版）

| 原Pillar | 价值增强版 |
|---------|---------|
| 真实性优先 | **心理真实性优先**：捕捉的不是"教室长什么样"，而是"坐在教室里是什么感觉" |
| 没有标准答案 | **没有标准答案，但有后果**：每种选择都有心理代价，游戏不评判，只呈现**心理代价** |
| 涌现叙事 | **涌现的同理心**：故事不是预设的，但系统会生成"你伤害了某人"或"你帮助了某人"的叙事 |
| 双重视角 | **权力与脆弱的双重视角**：不是"猫抓老鼠"，而是"两个被困在制度里的人" |

## 第二章：核心机制的价值化重构

### 2.1 精力系统 → 心理能量系统（Psychic Energy System）

**原设计**：精力=体力，用完了就休息
**价值版**：精力=心理能量，包括"意志力储备""情绪调节能力"

```
心理能量池：100点/回合，自动恢复5点/回合

消耗项：
- 专注学习：消耗"专注力"（不是体力，是注意力枯竭）
- 维持社会面具：消耗"情绪调节能力"
- 社交互动：消耗"社交能量"
- 压抑情绪：消耗"心理承受力"（高代价）

恢复项：
- 短暂走神：恢复5点
- 与同桌低声说话：恢复10点（社交支持）
- 看窗外：恢复8点（认知脱离）
- 深呼吸（主动操作）：恢复15点（正念机制）
```

**隐喻**：你不是身体累，你是注意力枯竭。连续学习不消耗"体力"，消耗"专注力"。

### 2.2 隐蔽值系统 → 社会面具系统（Social Mask System）

**原设计**：隐蔽值=被抓风险
**价值版**：隐蔽值=维持"好学生"形象的心理成本

```
社会面具成本：0-100
- 高面具成本 = 你在消耗大量心理能量维持伪装，这会加速崩溃
- 低面具成本 = 你"做自己"，但可能触发制度惩罚

机制：
- 每回合维持"认真学习"姿态：+3面具成本
- 被同学看到真实状态：-10面具成本，但关系+5（真实连接）
- 被老师看到真实状态：触发制度反应
- 面具成本>80：每回合额外消耗心理能量10点
```

**隐喻**：你不是在"躲避老师"，你是在计算"维持伪装的心理代价"。

### 2.3 关系系统 → 支持网络系统（Support Network System）

**原设计**：关系值=谁愿意帮你
**价值版**：关系值=你的社会支持网络

```
支持网络强度：0-100（每个关系独立计算）

功能重构：
- 孤独（支持值低）不是"没人帮你抄作业"，而是"崩溃时没人发现"
- 同桌支持>50：当你崩溃时，同桌会主动关心（触发保护事件）
- 前桌支持>30：可以"假装问题目"进行真实对话
- 后桌支持>20：可以传递情绪支持（递纸条"你还好吗"）

支持网络的作用：
- 心理能量恢复速度+（支持网络强度/10）%
- 崩溃阈值+（支持网络强度/5）
- 孤独事件触发概率-（支持网络强度/2）%
```

### 2.4 教师AI → 制度压力传导系统（Institutional Pressure System）

**原设计**：老师是"对手"
**价值版**：老师也是**被制度压迫的人**

```
教师状态：
- 学校KPI压力：影响巡视频率和严厉程度
- 家长投诉压力：影响对学生的容忍度
- 自身疲惫：影响决策质量
- 个人价值观：某些老师更关心学生心理健康

教师行为模式：
- 高压KPI + 疲惫：严厉、频繁巡视、零容忍
- 高压KPI + 精力充沛：严厉但公平
- 低KPI + 疲惫：放任、坐在讲台不动
- 低KPI + 关注心理：选择性执法、主动关心学生

教师视角揭示：
- 你今晚批评了3个学生。其中1个正在经历家庭问题。你不知道。
- 你选择性执法不是因为懒，是因为你不可能同时关注50个人。
- 那个你认为"在反抗你"的学生，实际上是在崩溃边缘。
```

### 2.5 结局系统的价值重构

| 原结局 | 价值增强版 |
|-------|---------|
| 学霸之夜 | "你完成了所有任务，但你的同桌崩溃了。你注意到吗？" + 解锁同桌视角回放 |
| 摸鱼大师 | "你成功逃避了制度，但逃避本身是一种消耗。你快乐吗？" + 心理能量消耗分析 |
| 身心俱疲 | "这不是失败。这是信号。你的角色需要休息，需要说话，需要帮助。" + 资源链接 |
| 崩溃边缘 | "触发资源链接：如果你或你认识的人正在经历类似感受，这里是中国心理援助热线..." |
| 社交之夜 | "你今晚没有完成很多作业，但你帮助了同桌度过焦虑发作。这很重要。" |
| 教师理解 | "你今晚选择性执法了。你保护了3个学生，但可能错过了1个真正需要帮助的人。" |

# 第二部分：3D第一视角核心设计

## 第三章：3D视角与感官牢笼

### 3.1 第一视角的设计哲学

晚自习的第一视角是一个**感官牢笼**：
- 你被固定在椅子上，视野有限
- 你知道老师可能在任何地方，但看不到
- 你的同桌可能是盟友，也可能是压力源
- 唯一能完全信任的是自己的耳朵

**核心能力**：感官管理（Sensory Management）
- 视觉注意力：看哪里？看多久？什么时候收回目光？
- 听觉空间化：脚步声从哪个方向来？距离多远？速度多快？
- 风险评估：基于不完整信息，判断当前行动是否安全

### 3.2 摄像机系统（Camera System）

#### 3.2.1 视角规格

| 参数 | 值 | 说明 |
|------|-----|------|
| 位置 | 固定于玩家座位 | 头部高度（课桌上方30cm） |
| 水平FOV | 100度 | 略宽于人眼自然视野（增强余光感） |
| 垂直FOV | 70度 | 抬头看黑板到低头看桌面 |
| 头部转动范围 | 水平±60度，垂直+30/-45度 | 超过范围显示"脖子僵硬"提示 |
| 转动速度 | 正常120度/秒，紧张时变慢 | 被老师注意时，转动有延迟 |
| 景深 | 聚焦距离1.5m，模糊范围0.5-3m | 模拟真实视觉，远处模糊 |
| 边缘暗角 | 屏幕边缘15%区域轻微变暗 | 集中注意力于中心 |

#### 3.2.2 视野分区

```
┌─────────────────────────────────────┐
│  [A区：上方远景] 黑板、讲台、时钟    │  ← 抬头看，需要转动>30度
│                                     │
│  [B区：中景] 前3排同学背影、过道    │  ← 正常视角，信息最丰富
│                                     │
│  [C区：近景] 桌面、双手、书本、手机  │  ← 低头看，执行行动区域
│                                     │
│  [D区：左侧余光] 同桌、窗户         │  ← 不转头，仅余光感知
│                                     │
│  [E区：右侧余光] 同桌、走廊方向     │  ← 不转头，仅余光感知
│                                     │
│  [F区：后方视野] 后门、走廊、身后   │  ← 需要明显回头，坐着时暴露大幅上升
│                                     │
└─────────────────────────────────────┘
```

| 区域 | 清晰度 | 信息类型 | 获取成本 |
|------|--------|---------|---------|
| A区（上方） | 高（聚焦） | 老师位置、黑板内容、时钟 | 需要抬头，暴露风险 |
| B区（中景） | 中（景深模糊） | 同学姿势、老师移动 | 正常视角，无额外成本 |
| C区（近景） | 极高（手边） | 自己的行动、物品状态 | 低头，暴露值-（老师看不到） |
| D/E区（余光） | 低（模糊+变形） | 颜色、运动、大体轮廓 | 无成本，但信息有限 |
| F区（后方） | 中 | 后门、走廊和身后动静 | 坐着回头成本极高；获准离座后成本降低 |

### 3.3 视觉注意力系统（Visual Attention System）

```
注意力池：100点/回合，自动恢复20点/回合

消耗：
- 看向A区（抬头）：消耗30点/秒
- 看向B区（正常）：消耗10点/秒
- 看向C区（低头）：消耗5点/秒
- 余光感知D/E区：消耗0点（被动）
- 回头看F区：消耗高，且未起立时显著增加暴露和压力

注意力耗尽：视野模糊，无法聚焦，必须低头休息

注意力与暴露值的关系：
- 注意力集中在某区域时，对该区域的感知+50%
- 注意力集中在C区（桌面）时，对B区（中景）的感知-50%
- 注意力集中在A区（讲台）时，对C区（桌面）的控制-30%
```

### 3.4 余光系统（Peripheral Vision）

余光是3D版的核心机制：

| 余光类型 | 触发条件 | 信息质量 | 示例 |
|---------|---------|---------|------|
| 颜色变化 | 余光区域出现新颜色 | 高 | 老师红色衣服进入右侧余光 |
| 运动检测 | 余光区域有快速移动 | 中 | 有人站起、老师快步走过 |
| 光影变化 | 光线突然变化 | 高 | 有人打开手机屏幕（蓝光） |
| 持续存在 | 余光区域有静止物体 | 低 | 同桌一直在旁边 |

**余光UI提示**：
- 屏幕边缘出现**方向性光晕**：左侧边缘泛红 = 左侧有异常
- 不显示具体是什么，只显示"方向和强度"
- 玩家必须转头确认，但转头有暴露风险

### 3.5 低头/抬头状态机

```
[正常视角] —— 头部水平，看向前方
    ├── 抬头（看讲台/黑板）—— 暴露值+5/秒，老师好感+（"认真"），但看不到桌面
    ├── 低头（看桌面/抽屉）—— 暴露值-5/秒，进入"隐蔽状态"，但看不到老师
    ├── 侧头（看同桌）—— 暴露值+3/秒，社交信息+，但引起同桌注意
    ├── 回头（看后门/走廊）—— 未获准离座时暴露值大幅增加，确认信息但代价高
    └── 固定（不动）—— 暴露值不变，注意力恢复
```

### 3.6 获准离座后的自由活动

当玩家通过“举手”获得老师允许后，进入 60 秒短时自由活动。为避免离座成为无限低成本恢复手段，晚自习每一节课最多允许举手离座一次；课间不受此限制，改为 5 分钟自由活动：

- 鼠标拖拽控制第一人称朝向。
- 按住空格沿当前朝向前进。
- 按住 Shift 侧身，通过桌椅间较窄的通道，速度降低。
- 前门和后门均可进入右侧走廊；走廊前后延伸，后门附近设置洗手间。
- 窗户保持视觉通透但不可通行，避免“从窗户跳出教室”。
- 靠近门时可开关门，门状态会保持到下一次离座。
- 靠近饮水机时可补满水杯；靠近洗手间时可进入并处理如厕需求。

### 3.7 身体需求拆分与交互闭环

旧版“身体需求”被拆分为三条独立压力线：

| 状态 | 范围 | 上升来源 | 处理方式 | 当前实现 |
|------|------|----------|----------|----------|
| 口渴 | 0-100 | 时间推进、晚自习疲惫倍率 | 喝水、饮水机补水 | 每次喝水口渴 -20，杯水 -25；杯水不足时需要去饮水机 |
| 饥饿 | 0-100 | 时间推进、学习消耗 | 吃零食、后续可扩展储物柜物品 | 吃零食降低饥饿，但产生包装声和暴露风险 |
| 如厕 | 0-100 | 时间推进、喝水后的长期压力可扩展 | 去洗手间 | 洗手间交互消耗 10 秒，如厕归 0 |

设计目的：

- 让“身体不是背景数值”，而是玩家必须安排时机处理的资源。
- 让课间具备游戏价值，而不仅是叙事段落。
- 让教师线后续可以围绕“是否允许学生处理身体需求”形成更真实的管理矛盾。

## 第四章：3D空间与教室布局

### 4.1 教室尺寸（真实比例）

```
教室尺寸：12m(长) × 8m(宽) × 3.5m(高)
SceneKit单位：1单位 = 1米

座位排列：
- 5排 × 4列 = 20个座位
- 排距：1.5m
- 列距：1.2m
- 过道：中间1.2m宽，两侧靠墙

玩家座位：第3排中间（3, 2）
- 坐标：(-0.6, 0.8, 1.5)
- 前方：第1-2排（4个同学背影）
- 后方：第4-5排（4个同学背影，需转头180度）
- 左侧：同桌（第3排左列）
- 右侧：同桌（第3排右列）

讲台位置：(0, 0.3, -4.5)
黑板位置：(0, 1.8, -5.2)
```

### 5.2 GameManager（游戏管理器）

```swift
import SceneKit
import SwiftUI
import Combine
import AVFoundation

/// 游戏全局状态枚举
public enum GameState: Equatable {
    case menu           // 主菜单
    case playing        // 游戏中
    case paused         // 暂停
    case eventCinematic // 事件演出中
    case discovery      // 被发现演出
    case ending         // 结局展示
    case teacherView    // 教师视角模式
}

/// 游戏回合阶段
public enum TurnPhase {
    case observation    // 观察阶段（自由转动视角）
    case action         // 行动阶段（选择行动）
    case resolution     // 结算阶段（执行结果）
    case teacherTurn    // 教师行动阶段
}

/// 核心游戏管理器，单例模式
public final class GameManager: ObservableObject {

    // MARK: - Singleton
    public static let shared = GameManager()

    // MARK: - Published Properties
    @Published public var gameState: GameState = .menu
    @Published public var currentTurn: Int = 0
    @Published public var maxTurns: Int = 27  // 3节课 × 9回合
    @Published public var currentPhase: TurnPhase = .observation
    @Published public var playerState: PlayerState = PlayerState()
    @Published public var teacherState: TeacherState = TeacherState()
    @Published public var isTeacherView: Bool = false

    // MARK: - Scene & Systems
    public private(set) var classroomScene: ClassroomScene?
    public private(set) var audioManager: SpatialAudioManager?
    public private(set) var eventSystem: EventSystem?
    public private(set) var teacherAI: TeacherAI3D?
    public private(set) var classmateAIs: [ClassmateAI3D] = []
    public private(set) var cameraController: FirstPersonCameraController?

    // MARK: - Cancellables
    private var cancellables = Set<AnyCancellable>()

    // MARK: - Initialization
    private init() {
        setupBindings()
    }

    // MARK: - Game Lifecycle

    /// 开始新游戏
    /// - Parameter isTeacherView: 是否以教师视角开始
    public func startGame(isTeacherView: Bool = false) {
        self.isTeacherView = isTeacherView
        self.gameState = .playing
        self.currentTurn = 0
        self.playerState = PlayerState()
        self.teacherState = TeacherState()

        // 初始化场景
        classroomScene = ClassroomScene()

        // 初始化系统
        audioManager = SpatialAudioManager()
        eventSystem = EventSystem()
        teacherAI = TeacherAI3D()

        // 初始化相机控制器
        if let cameraNode = classroomScene?.playerCameraNode {
            cameraController = FirstPersonCameraController(cameraNode: cameraNode)
        }

        // 生成同学
        generateClassmates()

        // 开始第一回合
        startTurn()
    }

    /// 开始新回合
    public func startTurn() {
        guard currentTurn < maxTurns else {
            triggerEnding()
            return
        }

        currentTurn += 1
        currentPhase = .observation

        // 更新所有系统
        updateSystems()

        // 检查触发条件
        checkTriggers()
    }

    /// 推进到行动阶段
    public func advanceToActionPhase() {
        currentPhase = .action
    }

    /// 执行玩家行动
    /// - Parameter action: 选择的行动
    public func executePlayerAction(_ action: PlayerAction) {
        currentPhase = .resolution
        playerState.currentAction = action

        // 计算行动结果
        let result = calculateActionResult(action)
        applyActionResult(result)

        // 推进到教师回合
        DispatchQueue.main.asyncAfter(deadline: .now() + 1.0) { [weak self] in
            self?.teacherTurn()
        }
    }

    /// 教师回合
    public func teacherTurn() {
        currentPhase = .teacherTurn
        teacherAI?.update()

        // 检查暴露
        if playerState.exposure >= 100 {
            triggerDiscovery()
            return
        }

        // 进入下一回合
        DispatchQueue.main.asyncAfter(deadline: .now() + 0.5) { [weak self] in
            self?.startTurn()
        }
    }

    /// 触发被发现演出
    public func triggerDiscovery() {
        gameState = .discovery
        audioManager?.playDiscoverySound()
        classroomScene?.playDiscoveryCinematic()
    }

    /// 触发结局
    public func triggerEnding() {
        gameState = .ending
        let ending = calculateEnding()
        // 展示结局...
    }

    /// 暂停/恢复游戏
    public func togglePause() {
        gameState = (gameState == .paused) ? .playing : .paused
    }

    // MARK: - Private Methods

    private func setupBindings() {
        // 监听玩家状态变化，触发对应音频
        $playerState
            .map { $0.exposure }
            .removeDuplicates()
            .sink { [weak self] exposure in
                self?.audioManager?.updateHeartbeat(intensity: exposure / 100.0)
            }
            .store(in: &cancellables)
    }

    private func updateSystems() {
        teacherAI?.update()
        classmateAIs.forEach { $0.update() }
        eventSystem?.checkTriggers()
        audioManager?.updateSpatialAudio()
    }

    private func checkTriggers() {
        // 检查崩溃阈值
        if playerState.psychicEnergy <= 0 {
            eventSystem?.triggerBreakdownEvent()
        }

        // 检查孤独事件
        if playerState.supportNetwork < 20 && Int.random(in: 0...100) < 15 {
            eventSystem?.triggerLonelinessEvent()
        }
    }

    private func generateClassmates() {
        classmateAIs = (0..<19).map { index in
            let position = calculateSeatPosition(index: index)
            let node = ProceduralCharacterGenerator.createStudent(seed: index)
            node.position = position
            classroomScene?.rootNode.addChildNode(node)
            return ClassmateAI3D(node: node, position: position, index: index)
        }
    }

    private func calculateSeatPosition(index: Int) -> SCNVector3 {
        let row = index / 4
        let col = index % 4
        return SCNVector3(
            Float(col) * 1.5 - 2.25,
            0,
            Float(row) * 1.5 - 3.0
        )
    }

    private func calculateActionResult(_ action: PlayerAction) -> ActionResult {
        // 详细计算逻辑...
        return ActionResult()
    }

    private func applyActionResult(_ result: ActionResult) {
        playerState.psychicEnergy += result.energyDelta
        playerState.exposure += result.exposureDelta
        playerState.maskCost += result.maskCostDelta
        playerState.homeworkProgress += result.homeworkDelta
    }

    private func calculateEnding() -> EndingType {
        // 基于多维度计算结局
        if playerState.psychicEnergy < 20 {
            return .breakdown
        } else if playerState.homeworkProgress > 80 && playerState.supportNetwork < 30 {
            return .academicButLonely
        } else if playerState.supportNetwork > 70 && playerState.homeworkProgress < 50 {

        }
        // ... 更多结局

    }
}
```

```swift
import SceneKit

/// 第一视角相机控制器，处理头部旋转和暴露值计算
public final class FirstPersonCameraController {

    // MARK: - Properties
    public let cameraNode: SCNNode
    private let headNode: SCNNode

    private var currentPitch: Float = 0    // 上下旋转（X轴）
    private var currentYaw: Float = 0    // 左右旋转（Y轴）

    private let maxPitch: Float = 0.8      // 约45度
    private let minPitch: Float = -1.2     // 约-70度（低头看桌面）
    private let maxYaw: Float = 1.0        // 约60度

    private var lastUpdateTime: TimeInterval = 0
    private var headDirection: HeadDirection = .forward

    // MARK: - Initialization
    public init(cameraNode: SCNNode) {
        self.cameraNode = cameraNode
        self.headNode = SCNNode()
        cameraNode.addChildNode(headNode)

        // 初始朝向
        headNode.eulerAngles = SCNVector3(0, 0, 0)
    }

    // MARK: - Rotation Control

    /// 更新旋转（基于输入增量）
    /// - Parameters:
    ///   - deltaYaw: 水平旋转增量（弧度）
    ///   - deltaPitch: 垂直旋转增量（弧度）
    public func updateRotation(deltaYaw: Float, deltaPitch: Float) {
        currentYaw += deltaYaw
        currentPitch += deltaPitch

        // 限制范围
        currentYaw = clamp(currentYaw, -maxYaw, maxYaw)
        currentPitch = clamp(currentPitch, minPitch, maxPitch)

        // 应用旋转
        headNode.eulerAngles = SCNVector3(currentPitch, currentYaw, 0)

        // 更新头部方向枚举
        updateHeadDirection()

        // 计算暴露值：转头速度
        let turnSpeed = abs(deltaYaw) + abs(deltaPitch)
        if turnSpeed > 0.1 {
            let exposureIncrease = turnSpeed * 5.0
            GameManager.shared.playerState.exposure += exposureIncrease
        }
    }

    /// 平滑看向目标
    /// - Parameter target: 目标世界坐标
    public func lookAt(_ target: SCNVector3) {
        let direction = target - cameraNode.worldPosition
        let targetYaw = atan2(direction.x, direction.z)
        let targetPitch = atan2(direction.y, sqrt(direction.x * direction.x + direction.z * direction.z))

        let action = SCNAction.rotateTo(
            x: CGFloat(targetPitch),
            y: CGFloat(targetYaw),
            z: 0,
            duration: 0.5
        )
        headNode.runAction(action)

        currentPitch = targetPitch
        currentYaw = targetYaw
        updateHeadDirection()
    }

    /// 强制抬头（被发现时）
    public func forceLookUp() {
        let action = SCNAction.rotateTo(x: 0.5, y: 0, z: 0, duration: 1.0)
        action.timingMode = .easeInEaseOut
        headNode.runAction(action)
        currentPitch = 0.5
        currentYaw = 0
        updateHeadDirection()
    }

    /// 获取当前头部方向
    public func getHeadDirection() -> HeadDirection {
        return headDirection
    }

    // MARK: - Private Methods

    private func updateHeadDirection() {
        if currentPitch > 0.3 {
            headDirection = .up
        } else if currentPitch < -0.6 {
            headDirection = .down
        } else if currentYaw > 0.5 {
            headDirection = .right
        } else if currentYaw < -0.5 {
            headDirection = .left
        } else {
            headDirection = .forward
        }
    }

    private func clamp<T: Comparable>(_ value: T, _ min: T, _ max: T) -> T {
        return Swift.min(Swift.max(value, min), max)
    }
}

public enum HeadDirection {
```

```swift
/// 教师AI，制度压力传导系统的核心
public final class TeacherAI3D {

    // MARK: - Properties
    public var position: SCNVector3 = SCNVector3(0, 0, -4.5)
    public var state: TeacherState = .atPodium
    public var targetPosition: SCNVector3?
    public var movementSpeed: Float = 1.0
    public var isMoving: Bool = false

    // 教师内部状态（制度压力）
    public var kpiPressure: Float = 50        // KPI压力
    public var fatigue: Float = 30            // 疲劳度
    public var empathy: Float = 40            // 同理心
    public var personalTolerance: Float = 50   // 个人容忍度

    // 观察目标
    public var currentAttentionTarget: Int? = nil
    public var suspicionLevels: [Int: Float] = [:]  // 对每个学生的怀疑值

    // 移动路径
    private var patrolPath: [SCNVector3] = []
    private var currentPathIndex: Int = 0

    // MARK: - Update Loop

    public func update() {
        updateInternalState()

        switch state {
        case .atPodium:
            updateAtPodium()
        case .patrolling:
            updatePatrolling()
        case .movingToStudent:
            updateMovingToStudent()
        case .observing:
            updateObserving()
        case .resting:
            updateResting()
        case .rearDoorObserving:
            updateRearDoorObserving()
        case .fakePatrolling:
            updateFakePatrolling()
        }

        // 更新空间音频位置
        updateAudioPosition()

        // 检查是否能看到玩家
        checkPlayerVisibility()
    }

    // MARK: - State Updates

    private func updateAtPodium() {
        let roll = Float.random(in: 0...1)

        if roll < 0.25 {
            // 开始巡视
            state = .patrolling
            generatePatrolPath()
        } else if roll < 0.4 && fatigue > 60 {
            // 休息
            state = .resting
        } else if roll < 0.5 && kpiPressure > 70 {
            // 后门观察（高压下的严格行为）
            state = .rearDoorObserving
            targetPosition = SCNVector3(-4, 0, 3.5)
        }
    }

    private func updatePatrolling() {
        guard !patrolPath.isEmpty else {
            state = .atPodium
            return
        }

        let target = patrolPath[currentPathIndex]
        moveTowards(target)

        if distance(position, target) < 0.2 {
            currentPathIndex += 1
            if currentPathIndex >= patrolPath.count {
                state = .atPodium
                patrolPath.removeAll()
            } else {
                // 随机在某个点停下观察
                if Float.random(in: 0...1) < 0.3 {
                    state = .observing
                }
            }
        }
    }

    private func updateMovingToStudent() {
        guard let target = targetPosition else {
            state = .atPodium
            return
        }

        moveTowards(target, speed: 2.0)

        if distance(position, target) < 0.5 {
            state = .observing
            // 触发对学生的观察事件
        }
    }

    private func updateObserving() {
        // 停留2-5秒
        DispatchQueue.main.asyncAfter(deadline: .now() + Double.random(in: 2...5)) { [weak self] in
            self?.state = .patrolling
        }
    }

    private func updateResting() {
        fatigue = max(0, fatigue - 5)
        if fatigue < 30 {
            state = .atPodium
        }
    }

    private func updateRearDoorObserving() {
        // 无声观察，从后门看教室
        // 如果玩家此时违规且面向后方，可能看到老师
        position = SCNVector3(-4, 0, 3.5)

        // 5-10秒后离开
        DispatchQueue.main.asyncAfter(deadline: .now() + Double.random(in: 5...10)) { [weak self] in
            self?.state = .atPodium
        }
    }

    private func updateFakePatrolling() {
        // 故意制造脚步声后停下
        // 玩家会听到脚步声接近，但老师实际上在某处停下
    }

    // MARK: - Movement

    private func moveTowards(_ target: SCNVector3, speed: Float? = nil) {
        let direction = target - position
        let dist = distance(position, target)

        guard dist > 0.1 else { return }

        let normalized = direction / dist
        let moveSpeed = speed ?? movementSpeed
        position += normalized * moveSpeed * 0.016  // 假设60fps

        isMoving = true
    }

    private func generatePatrolPath() {
        // 生成过道的随机路径点
        let pathCount = Int.random(in: 3...6)
        patrolPath = (0..<pathCount).map { _ in
            SCNVector3(
                Float.random(in: -2...2),
                0,
                Float.random(in: -3...3)
            )
        }
        currentPathIndex = 0
    }

    // MARK: - Vision System

    private func checkPlayerVisibility() {
        guard let playerPos = GameManager.shared.classroomScene?.playerCameraNode?.worldPosition else { return }

        let dist = distance(position, playerPos)
        guard dist < 8.0 else { return }

        // 计算角度
        let toPlayer = playerPos - position
        let forward = SCNVector3(0, 0, 1)  // 简化，实际需要老师朝向
        let angle = angleBetween(forward, toPlayer)

        guard angle < 60.0 else { return }  // 视野锥120度

        // 射线检测（简化版）
        // 实际应使用scene.physicsWorld.rayTest

        // 计算暴露值
        var exposure: Float = 10.0
        exposure *= (1.0 - Float(dist) / 8.0)

        if currentAttentionTarget == -1 {  // -1表示玩家
            exposure *= 2.0
        }

        GameManager.shared.playerState.exposure += exposure
    }

    // MARK: - Internal State

    private func updateInternalState() {
        // 疲劳度随时间增加
        fatigue = min(100, fatigue + 0.5)

        // KPI压力根据回合数变化
        let turn = GameManager.shared.currentTurn
        if turn > 18 {  // 第三节
            kpiPressure = min(100, kpiPressure + 1)
        }
    }

    // MARK: - Audio

        GameManager.shared.audioManager?.updateSourcePosition(
            id: "teacher_footsteps",
            position: position
        )
    }

    /// 获取教师视角描述（用于教师模式）
    public func getTeacherPerspectiveDescription() -> String {
        var desc = "你感到"
        if fatigue > 70 { desc += "极度疲惫，" }
        else if fatigue > 40 { desc += "有些疲惫，" }

        if kpiPressure > 70 { desc += "学校要求今晚必须保持纪律。" }
        else { desc += "今晚的KPI压力不大。" }

        desc += "\n你注意到教室里有"
        let suspiciousCount = suspicionLevels.filter { $0.value > 50 }.count
        if suspiciousCount > 0 {
            desc += "\(suspiciousCount)个学生看起来不太对劲。"
        } else {
            desc += "学生们似乎都在认真学习。"
        }

        return desc
    }
```

```swift
/// 程序化角色生成器，完全用代码生成角色，无外部模型依赖

// MARK: - ProceduralStudent

public final class ProceduralStudent {

    // MARK: - Parameters
    private let seed: Int
    private var rng: SeededRandomGenerator

    public var height: Float
    public var bodyType: BodyType
    public var skinColor: UIColor
    public var uniformColor: UIColor
    public var hairStyle: HairStyle
    public var hairColor: UIColor
    public var glasses: Bool
    public var posture: Posture

    // MARK: - Initialization
    public init(seed: Int) {
        self.seed = seed
        self.rng = SeededRandomGenerator(seed: seed)

        // 基于seed生成参数
        self.height = rng.random(in: 1.5...1.8)
        self.bodyType = BodyType.allCases[Int(rng.random(in: 0...2))]
        self.skinColor = SkinColor.allCases[Int(rng.random(in: 0...4))].uiColor
        self.uniformColor = UniformColor.allCases[Int(rng.random(in: 0...7))].uiColor
        self.hairStyle = HairStyle.allCases[Int(rng.random(in: 0...4))]
        self.hairColor = HairColor.allCases[Int(rng.random(in: 0...3))].uiColor
        self.glasses = rng.random(in: 0...1) > 0.7
        self.posture = Posture.allCases[Int(rng.random(in: 0...3))]
    }

    // MARK: - Generation

    public func generate() -> SCNNode {
        let root = SCNNode()
        root.name = "student_\(seed)"

        // 整体缩放（基于身高）
        let scale = height / 1.7
        root.scale = SCNVector3(scale, scale, scale)

        // 1. 臀部/骨盆（根节点）
        let hips = SCNNode()
        hips.position.y = 0.9  // 坐姿臀部高度
        root.addChildNode(hips)

        // 2. 躯干
        let torsoWidth: CGFloat = bodyType == .skinny ? 0.3 : (bodyType == .athletic ? 0.4 : 0.45)
        let torso = SCNBox(width: torsoWidth, height: 0.5, length: 0.2, chamferRadius: 0.02)
        torso.firstMaterial = createUniformMaterial()
        let torsoNode = SCNNode(geometry: torso)
        hips.addChildNode(torsoNode)

        // 3. 头部
        let head = SCNSphere(radius: 0.12)
        head.firstMaterial = createSkinMaterial()
        let headNode = SCNNode(geometry: head)
        headNode.position.y = 0.35
        torsoNode.addChildNode(headNode)

        // 4. 头发
        let hairNode = createHair(style: hairStyle, color: hairColor)
        hairNode.position.y = 0.02
        headNode.addChildNode(hairNode)

        // 5. 眼睛（简单平面）
        let eyeLeft = createEye()
        eyeLeft.position = SCNVector3(-0.04, 0.02, 0.11)
        headNode.addChildNode(eyeLeft)

        let eyeRight = createEye()
        eyeRight.position = SCNVector3(0.04, 0.02, 0.11)
        headNode.addChildNode(eyeRight)

        // 6. 眼镜
        if glasses {
            let glassesNode = createGlasses()
            glassesNode.position.y = 0.02
            headNode.addChildNode(glassesNode)
        }

        // 7. 手臂
        let armL = createArm(isLeft: true)
        armL.position = SCNVector3(-Float(torsoWidth)/2 - 0.05, 0, 0)
        torsoNode.addChildNode(armL)

        let armR = createArm(isLeft: false)
        armR.position = SCNVector3(Float(torsoWidth)/2 + 0.05, 0, 0)
        torsoNode.addChildNode(armR)

        // 8. 腿（坐姿）
        let legL = createLeg(isLeft: true)
        legL.position = SCNVector3(-0.15, -0.25, 0.2)
        hips.addChildNode(legL)

        let legR = createLeg(isLeft: false)
        legR.position = SCNVector3(0.15, -0.25, 0.2)
        hips.addChildNode(legR)

        // 9. 应用姿势
        applyPosture(to: root)

        // 10. 桌面物品（装饰）
        let deskItems = createDeskItems()
        deskItems.position = SCNVector3(0, 0.75, 0.2)
        root.addChildNode(deskItems)

        return root
    }

        mat.diffuse.contents = uniformColor
        mat.roughness.contents = 0.6
        mat.metalness.contents = 0.0
        return mat
    }

    private func createSkinMaterial() -> SCNMaterial {

    private func createHair(style: HairStyle, color: UIColor) -> SCNNode {
        let root = SCNNode()

        let hairMat = SCNMaterial()
        hairMat.diffuse.contents = color
        hairMat.roughness.contents = 0.9

            top.firstMaterial = hairMat
            let topNode = SCNNode(geometry: top)
            topNode.scale.y = 0.6
            root.addChildNode(topNode)

            back.firstMaterial = hairMat
            let backNode = SCNNode(geometry: back)
            backNode.position = SCNVector3(0, -0.15, -0.1)

            topNode.scale.y = 0.6

            let tailNode = SCNNode(geometry: tail)

        case .sideSitting:

    }

    var length: Float {

public func distance(_ a: SCNVector3, _ b: SCNVector3) -> Float {
    return (a - b).length

public func angleBetween(_ a: SCNVector3, _ b: SCNVector3) -> Float {
    let dot = a.x * b.x + a.y * b.y + a.z * b.z
    let lenA = a.length
    let lenB = b.length
    guard lenA > 0 && lenB > 0 else { return 0 }
    return acos(dot / (lenA * lenB)) * 180 / Float.pi
}
```

```swift
/// 空间音频管理器，处理所有3D定位音频

    private func setupAudioEngine() {
        audioEngine = AVAudioEngine()

        environment = AVAudioEnvironmentNode()
        environment.reverbParameters.enable = true
        environment.reverbParameters.loadFactoryReverbPreset(.mediumRoom)

        audioEngine.connect(environment, to: audioEngine.mainMixerNode, format: nil)

    private func loadAudioFile(named: String) -> AVAudioPCMBuffer? {
        guard let url = Bundle.main.url(forResource: named, withExtension: "wav") ??
                        Bundle.main.url(forResource: named, withExtension: "mp3") else {
            print("Warning: Audio file not found: \(named)")
            return nil
        }

        do {
            let file = try AVAudioFile(forReading: url)
            let format = file.processingFormat
            let frameCount = AVAudioFrameCount(file.length)
            guard let buffer = AVAudioPCMBuffer(pcmFormat: format, frameCapacity: frameCount) else {
                return nil
            }
            try file.read(into: buffer)
            return buffer
        } catch {
            print("Error loading audio file \(named): \(error)")
            return nil
        }

    public func addSpatialSource(
        id: String,
        audioName: String,
        position: SCNVector3,
        loops: Bool = false,
        reverbBlend: Float = 0.3
    ) {
        guard let buffer = audioBuffers[audioName] else {
            print("Warning: Audio buffer not found for \(audioName)")
            return

        player.position = AVAudio3DPoint(x: position.x, y: position.y, z: position.z)

        player.pointSourceInHeadMode = .mono

        audioEngine.connect(player, to: environment, format: buffer.format)

        } else {
            player.scheduleBuffer(buffer, at: nil, options: [], completionHandler: nil)

        player.play()

            let player = AVAudioPlayerNode()
            audioEngine.attach(player)

            player.scheduleBuffer(buffer, at: nil, options: .loops, completionHandler: nil)
            player.play()
```

```swift
func updatePsychicEnergyAudio() {
    let energy = GameManager.shared.playerState.psychicEnergy

func updateMaskVisuals() {

func checkSupportNetworkEvents() {
    let support = GameManager.shared.playerState.supportNetwork

    }
```

### 9.3 比赛展示价值陈述

**Elevator Pitch**：
> "《晚自习模拟器》是一款关于中国高中生心理健康的沉浸式体验游戏。玩家通过第一视角经历一个普通的晚自习，管理自己的心理能量、社交关系、和制度压力。游戏的双重视角让学生理解'原来老师也不容易'，让教师理解'原来学生不是故意作对'。我们的目标不是娱乐，而是**构建一个同理心引擎**——让不同立场的人，在体验中理解彼此。"

**评委常见问题**：

**Q: 这游戏能真正帮助心理健康吗？**
A: 游戏本身不是治疗工具，但它是**认知入口**。通过游戏，学生学会识别"焦虑信号"（心跳加速、无法专注、想逃离），这些信号在真实生活中被忽视。游戏结束后，我们提供**真实资源链接**（学校心理咨询、公益热线）。更重要的是，游戏创造了一个**安全的谈论空间**——"我昨晚玩了这个游戏，我的角色崩溃了，这让我想到我自己..."

**Q: 为什么用游戏而不是直接做心理健康App？**
A: 直接的教育App会被学生抵触（"又一个说教软件"）。游戏的**沉浸性和选择性**让玩家主动探索："我想试试如果今晚不写作业会发生什么？"这种主动探索比被动接受信息更有效。而且，游戏让**教师和家长**也愿意参与——他们不是"被教育"的对象，而是"体验者"。
