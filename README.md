# 晚自习模拟器 3D Swift 垂直切片

这是根据 `晚自习模拟器_v4.0_完整设计文档.md` 搭建的 macOS SwiftUI + SceneKit 版本。

界面控件使用 macOS 26 / SwiftUI 官方 Liquid Glass 接口，包括 `glassEffect` 和官方玻璃按钮样式。

## 运行

```bash
swift run LateStudySimulator
```

当前环境只有 Command Line Tools，不能使用 `xcodebuild`。本项目用 Swift Package 组织，可以直接通过 `swift build` / `swift run` 构建运行。

项目目标平台为 macOS 26，需要支持 Liquid Glass API 的 SDK/运行环境。

本仓库只提交源码、资源占位和文档；`.app`、DMG、zip 等构建产物不入库。当前本地可通过 Swift Package 构建出 `LateStudySimulator` 可执行程序。

## 已实现范围

- SceneKit 3D 第一视角教室，固定第三排座位，约 100 度 FOV、景深和暗角。
- 12m x 8m 教室、5 排 x 4 列座位、19 名同学、1 名教师和玩家桌面物品。
- 教室空间包含前门、后门、右侧贯通走廊、前后班级门牌、后门附近洗手间、一班前方左转门厅和户外台阶视野；学生获批离座或课间自由活动后可通过前门或后门进入走廊。
- SwiftUI HUD：心理能量、社会面具、支持网络、压力、暴露、作业进度、口渴、杯水、饥饿值、如厕需求。
- 控件与 HUD 面板调用 SwiftUI 官方 Liquid Glass 接口，按钮使用官方玻璃按钮语义，设置里的开关和滑块行也统一调用 `glassEffect`。
- 开局菜单：进入晚自习前可先调整“如果模式”的时长、排名压力、巡视频率和交流规则。
- 学生视角支持鼠标拖拽环视，包含前方、低头、抬头、左侧、右侧和后方；坐着回头会显著增加暴露、压力和注意力成本。
- 自由活动输入：晚自习举手获准离座后进入 60 秒自由活动，且每节课最多一次；课间可进入 5 分钟自由活动。按住空格朝当前视角前进，按住 Shift 侧身通过窄缝，可提前回座。
- 固定座位与合法离座并存：未获准时仍以座位感官管理为主；获准后可通过轻量碰撞系统在教室、前/后门和走廊之间移动。
- 视觉注意力系统：不同视野区消耗不同注意力，低注意力会增加压力并降低聚焦质量；高压、低注意力或老师靠近时转头会变慢，并触发脖子僵硬/转头成本反馈。
- 3D 相机疲劳反馈：注意力下降时景深、暗角和饱和度会变化，模拟视野发散。
- 行动系统：写作业、看手机、传纸条、观察、同桌、深呼吸、看窗外、喝水、吃零食、举手离座。
- 教师巡逻、余光提示、被发现事件、崩溃教育时刻和多结局。
- 3D 教师视线反馈：老师节点带有制度压力视线锥和压力光源，靠近玩家、高 KPI 或后门观察时会更明显。
- 学生线/教师线分离：学生模式固定学生视角，教师模式固定班主任视角；结算页“返回菜单”后重新选择角色。
- 真实 3D 班主任视角：教师模式下相机会移动到老师当前位置，可从讲台、过道、后门或目标桌边观察。
- 教师模式操作：看全班、观察学生、公开提醒、低声提醒、关心询问、允许离开、选择性放过、坐下休息，体现选择性执法和制度压力。
- 教师后门无声观察：高 KPI 压力下老师可能从后门观察，学生听不到脚步，只能从后方细微信号感到不确定性。
- 教师假巡视：老师可以制造脚步声后停在远处，学生无法确认位置，听觉不确定性会推高压力和暴露判断成本。
- 学生视角真相回放：结局后可翻看每回合的表面现象、内心真实和教师误读/理解。
- 独立教师结局：根据教师提醒、关心、制度压力和班级风险生成“教师理解/制度压力传导”等结局。
- 三方同理心结局反思：结局报告会分别从学生、教师、家长视角解释同一晚的压力信号。
- 结局分析报告：展示焦虑峰值、心理消耗、面具负荷、支持缓冲、班级风险等数据，并用夜间轨迹图回放压力、能量和身体需求走势。
- 真实故事出口：结局根据玩家路径生成访谈式短篇和反思问题，把数据报告连接到心理教育叙事。
- 心理支持资源：结局内嵌中国心理援助热线、北京/上海心理援助热线和学校咨询提示。
- 同桌与 NPC 事件：焦虑、崩溃、关心你、掩护、手机、困倦等状态。
- 同学个体档案：每个同学有合作、守序、叛逆、同理心、焦虑基线和面具强度，影响压力漂移、违规倾向和是否提供帮助。
- 3D 个体差异：同学外观会根据档案生成不同体态、校服色、发型、眼镜、桌面纸张/便利贴/纸巾和手机可见度。
- 同桌行为检测：玩家看手机时，同桌会按性格作出举报、跟风、掩护或眼神提醒，并累积对玩家的怀疑值。
- 对视互动：切到左/右余光视角时会与对应同桌短暂对视，影响关系、支持、面具成本和怀疑值，并触发 3D 转头反馈。
- 风险原因呈现：教师视角、同桌条和回放会显示高风险学生的可能原因，而不只显示“纪律异常”。
- 程序化 NPC 动画：写字摆动、看手机蓝光、焦虑抖动、睡觉趴下、哭泣颤动、关心前倾、掩护侧身。
- 3D 小道具反馈：学生桌面有纸张/手机，玩家桌面手机带蓝色点光源，手机事件会增强亮度。
- 3D 桌面隐蔽反馈：低头、看手机、传纸条或吃零食时玩家抽屉会微开，抽屉缝隙、桌边摩擦声和手部姿态共同表现 C 区近景/隐蔽状态。
- 身体需求系统：口渴、饥饿和如厕需求会随晚自习推进上升，过高时触发身体信号、压力和注意力损耗；喝水会消耗水杯容量，饮水机可补满水杯，吃零食降低饥饿，进入洗手间消耗 10 秒并清空如厕需求。
- 3D 身体反馈：玩家桌面有零食包装、身体需求指示和腿部/座椅焦躁动作，口渴、饥饿、如厕需求或包装纸声会带动手部与坐姿变化。
- 3D 环境信息物件：黑板文字、单调前进的墙钟指针、玩家桌面作业纸、笔、进度条和手部会随时间、作业、压力与手机状态更新。
- 3D 写作业反馈：执行写作业时会显示作业纸、笔和右手写字动作；低头时可看到自己的双手与桌面物品。
- 3D 时间氛围：左侧窗户可看到户外白天/黑夜、太阳/月亮、云雨和路灯变化；右侧窗户可看到走廊、柜子和灯光。
- 3D 吊扇反馈：教室顶部吊扇从开局即低速旋转，后续时段逐渐加快，和吊扇底噪形成同步反馈。
- 事件日志：被发现、老师关心、崩溃信号、短暂停电、支持网络保护。
- 结构化事件选择：被发现、老师关心、玩家崩溃、同桌崩溃、支持网络保护、短暂停电都会给出不同选择和后果。
- 扩展事件池：新增手机通知、广播通知和后门敲门事件，分别体现社交连接诱惑、制度声音干预和不确定性压力，并影响压力、暴露、支持和教师 KPI。
- 事件演出层：崩溃、孤独、停电、被发现、同桌哭泣等事件会触发不同暗角、色彩、模糊和脉冲反馈，并显示心理教育提示。
- 同桌崩溃互动：可选择安慰、递纸巾、告诉老师、假装没看见，影响关系、支持网络、暴露和同桌压力。
- NPC 主动事件链：高压同学可能通过草稿纸低声求助，守序同学也可能犹豫举报，玩家可选择回应、共同调节、承认疲惫、压制或收手。
- NPC 跨局记忆：结局会保存同学关系、真相分享、压力余波和怀疑值；重新开始时会影响同学初始关系、压力和同桌条“有记忆”标记。
- 记忆落盘与清除：同学记忆会通过 `UserDefaults` 保留到下次启动，开局/设置面板提供 Liquid Glass 清除入口。
- 长线关系文本：同桌条会解释“有记忆”的具体含义，结局报告新增“关系余波”面板，回顾哪些同学会把这一晚带到下一次。
- 跨局事件变体：上一局建立信任的同学可能更早递来支持纸条，上一局留下怀疑的同学会更早观察和试探玩家。
- NPC 分支叙事：求助、举报、跨局信任和跨局怀疑事件会根据同学的高焦虑、高同理心、守序、叛逆或面具强度生成不同文本。
- “如果...”制度参数：可调整晚自习时长、是否允许低声交流、排名压力、教师巡视频率，并按当前设置重新开始。
- 制度参数会影响最大回合数、初始压力、教师 KPI、巡视速度、交流暴露惩罚、同学压力漂移和结局分析。
- 晚自习时间轴：HUD 显示 18:30 起算的动态时钟和第一节/课间/第二节/第三节阶段。
- 阶段影响：课间恢复心理能量和注意力，第三节提升疲劳与压力，并触发吊扇/灯管听觉线索。
- AVAudioEngine 程序化心跳、预警音和环境底噪，会根据心理能量、压力、支持网络、教师距离、阶段、灯光和教室噪声变化。
- 程序化听觉线索：脚步、纸张、手机、低语、椅子、抽泣、灯管、心跳、老师咳嗽/叹气等事件会产生方向、强度和短促音频提示。
- 程序化环境声：持续生成灯管嗡声、笔尖纸面噪声、远处车声和第三节吊扇底噪，不依赖外部音频素材。
- 可选真实音频素材：`Sources/LateStudySimulator/Resources/AudioCues` 或 `~/Library/Application Support/LateStudySimulator/AudioCues` 中放入同名 wav/mp3/m4a/aiff/caf 文件后，事件短音会优先播放真实素材；缺失时自动回退程序化声音。
- 可选真实循环环境声：`Sources/LateStudySimulator/Resources/AudioLoops` 或 `~/Library/Application Support/LateStudySimulator/AudioLoops` 支持 `light_hum`、`pen_scratch`、`ceiling_fan`、`outside_night` 循环素材，音量会随教室噪声、阶段、灯光和夜深程度变化。
- 音频素材覆盖率：开局/设置面板会显示真实事件短音和循环环境音的接入数量、缺失数量，悬停可查看具体缺失文件名，并提供脚步/心跳/纸张试听、刷新按钮和文件夹按钮打开外部素材目录。
- 3D 定位音频提示：事件短音会映射到教室坐标并接入 `AVAudioEnvironmentNode`，脚步、低语、桌面、头顶和窗外声音从对应方向出现。
- HUD 听觉面板：显示最近 3 条听觉线索，并用声场雷达呈现方向和强度，突出“70% 信息来自音频”的设计目标。
- 内心独白队列：玩家行动、风险上升和压力事件会生成最近独白，HUD 会展示角色此刻的主观感受。
- 孤独事件：当支持网络过低且压力升高时，会触发一次“教室坐满人但异常孤独”的选择事件，引导深呼吸、写下真实感受或继续维持面具。
